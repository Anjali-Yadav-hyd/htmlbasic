HTML BASICS

  1. What is DOCTYPE?

    It describes type of version we are using.

  2. What is the difference between paired tag and unpaired tag?

    Paired tags have closing tag (example include h1,p,a,img,i,b,etc.) while Unpaired tags do not have any closing tags (e.g: hr,br)

  3. What is the difference between block level and inline elements?

    Block Level Elements occupy entire row while Inline Elements occupy the space only the text ,viz. written, required.

  4. What are Schematic tags?

    Schematics tags have special meaning e.g: section,strong,header,footer,article,address,mark,em,etc.

  5. What is the difference between and b and strong tag?

    strong is a schematic tag while b is a non-schematic tag. strong tag is also used for blind person

  6. What is the difference between and i and em tag?

    em is a schematic tag while i is a non-schematic tag.

  7. What is the difference between and u and ins tag?

    The u tag underlines text without any additional styling, while the ins tag can be styled with CSS to indicate the inserted text, such as with a different color or background.

  8. What is the difference between s and del tag?

    The 's' represents contents that are no longer accurate or no longer relevant. The 'del' element represents a removal from the document.

  9. Strike Tag

    The 'strike' tag was used to define strikethrough text.

  10. What is the difference between mark and span tag?

    The 'mark' tag defines text that should be marked or highlighted. The 'span' tag is an inline container used to mark up a part of a text, or a part of a document. mark is a schematic tag while span is non-schematic.
