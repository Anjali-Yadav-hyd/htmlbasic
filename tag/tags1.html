HTML BASICS

  1. What is the difference between sup and sub tag?

    sup is used for superscript. sub is used for subscript.

  2. What is the difference between pre and code tag?

    pre and code are combined tags.'code' is a schematic tag.

  3. What is the difference between small and big tag?

    'big' tag was used to define bigger text.It is deprecated. 'small' tag is used for smaller text.

  2. What are the attributes present in 'abbr' tag and 'acronym' tag?

    Both are used for abbrevation. 'acronym' is deprecated tag

  3. What is the difference between absolute and relative path?

    Relative path links to an image that is hosted within the website. Absolute path links to an external image that is hosted on another website.

    src, alt, width, height are some attributes of img tag.

    src, auotplay, muted, controls, loop, preload are attributes of audio tag.
