HTML BASICS

  1. What are the uses of iframe tag?

    An inline frame is used to embed another document within the current HTML document.

    src, allowfullscreen, width, height, loading(eager,lazy), frameborder, sandbox are some attributes of iframe tag.

  2. What are the uses of anchor tag?

    The 'a' tag defines a hyperlink, which is used to link from one page to another.

    href, target(_blank,_self,_parent,_top), type, media are some attributes of a tag.

  3. What is the difference between ol and ul tag? What are their attributes?

    'ol' is used for ordered list and 'ul' is used for unordered list.

    reversed, type(A, 1, I, a, i), start are attributes of ol tag.

    type(circle, square, disc) is a attribute of ul tag

  4. What is the purpose of dl, dt and dd tag?

    dd tag is used in conjuction with dl and dt tag.

    The 'dd' tag is used to describe a term/name in a description list.

    The 'dl' tag defines a description list.
